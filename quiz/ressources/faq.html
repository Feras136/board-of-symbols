Typische Fragen und Antworten

  • Kommen immer die selben Fragen?
    Jein. Die meisten Fragen sind fest, aber bei einigen werden auch Werte variiert.
  • Kann man eigene Themen oder Fragen anlegen?
    Ja. Man muss nur entsprechende Dateien anlegen und bearbeiten. Beim Start durchsucht BoSQuiz das Verzeichnis ressources und baut aus den dort gefundenen .top Dateien das Themen-Menü auf.
  • Müsste das Verzeichnis nicht eigentlich resources heißen?
    Ja. Im Deutschen schreibt man Ressource, aber wenn der Plural Englisch ist, sollte das zweite s fehlen.
  • Ich habe eine Frage, einen Vorschlag, ...
    Entweder auf GitHub eintragen oder Stephan Euler ansprechen
